# RICE Images Agent Guide

This directory owns editorial image masters and generated derivatives.

- Read `../AGENTS.md`, `../ONTOLOGY.md`, and `MEMORY.md` before edits.
- Change canonical masters or prompt/provenance records first.
- Do not hand-edit generated `web/` or `thumb/` derivatives.
- Generated archival images must be described as visual reconstructions, not authenticated historical records.

Closeout:

- Run the asset build/check pipeline after image source changes.
- Append a short entry to this directory's `MEMORY.md` and to `../MEMORY.md`.
